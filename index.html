FAQ

Questions fréquentes

Tout ce que vous devez savoir avant de réserver votre voyage au Vietnam.

Comment réserver un circuit ?
Choisissez votre circuit, remplissez le formulaire de réservation, et notre équipe vous contacte sous 24h avec un devis personnalisé. Aucun paiement n'est demandé avant validation.
Quels sont les moyens de paiement acceptés ?
Nous acceptons PayPal, Stripe (carte bancaire) et 2Checkout. Tous les paiements sont sécurisés et chiffrés.
Puis-je personnaliser un circuit ?
Absolument. Chaque circuit est sur mesure. Notre équipe adapte les activités, l'hébergement et la durée selon vos préférences.
Quel est le délai d'annulation ?
Annulation gratuite jusqu'à 30 jours avant le départ. Entre 15 et 30 jours : 50% de remboursement. Moins de 15 jours : non remboursable.
Faut-il un visa pour le Vietnam ?
Les ressortissants français, belges et suisses bénéficient de l'exemption de visa pour 45 jours. Au-delà ou pour d'autres nationalités, notre équipe vous accompagne dans les démarches.

Administration

Comment gérer le site

Plateforme clé en main — voici comment recevoir les données et personnaliser le contenu.

Où vont les réservations ?

Les formulaires (contact, réservation, newsletter) sont actuellement affichés dans la console du navigateur et via des toasts de confirmation.

F12 → Console → Booking/Contact data

Connecter un backend

Pour recevoir les données par email ou dans une base de données, remplace les fonctions submitBooking et submitContact dans script.js :

fetch('/api/booking', { method:'POST', body: JSON.stringify(data) })

Solutions simples : Formspree (gratuit), EmailJS, ou un script PHP.

Modifier les circuits

Clique sur le bouton ⚙️ en bas à droite (admin toggle) pour éditer les titres, prix, et durées en mémoire.

script.js → const tours = [...]

Les données sont stockées dans le tableau JS. Modifie-le directement pour des changements permanents.

Changer les images

Toutes les images viennent d'Unsplash. Remplace les URLs dans les tableaux destinations, tours, et blogPosts dans script.js.

img: 'https://images.unsplash.com/...'

Traductions & Devises

Les sélecteurs en haut à gauche (FR/EN/VI + USD/EUR/VND) sont fonctionnels. Ajoute des traductions dans les tableaux de données.

titleFr, titleVi, nameFr, nameVi

Mettre en ligne

Héberge gratuitement sur GitHub Pages, Netlify, ou Vercel. Push les 7 fichiers du dossier et c'est en ligne !

git push origin main

Aucune configuration serveur nécessaire — site 100% statique.
